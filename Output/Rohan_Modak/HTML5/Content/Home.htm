Frequently Asked Questions

How to create Project?

Starting a new project in ContentMagic is quick and easy. Here are the steps you need to follow:

  1. Log in to your ContentMagic account. If you don't have an account yet, you can sign up for one on the ContentMagic website.

  2. Once you're logged in, click on the New Project button on the dashboard.

  3. Enter a name for your project and choose the type of content you want to create (e.g., blog post, social media post, video, etc.).

How to Import Content
ContentMagic allows you to easily import content from other projects into a new project. Here are the steps you can follow to import other ContentMagic projects into a new project:

  1. Log in to your ContentMagic account and create a new project.

  2. Click on the Import button in the project dashboard.

  3. Select the project you want to import content from.

  4. Choose the content you want to import: You can choose to import all the content from the project, or select specific content types, such as blog posts, social media posts, or videos.

  5. Review the imported content: Once you have selected the content you want to import, review it to ensure that it is correct and complete.

  6. Edit the imported content (if needed): If you need to make any changes to the imported content, you can do so using ContentMagic's editing tools.

  7. Save the imported content to your new project: Once you have reviewed and edited the imported content, save it to your new project. The imported content will now be available in your new project.
